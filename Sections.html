The two main groups organized by CuteSec are DC217 and CU2600.

Each group has its own webpage, click the link to find out more.


DC217

DC217 is the local Def-Con group for Champaign-Urbana. Def-Con is a yearly Hacking convention.
As such, this group focuses on cyber security / computers / Hacking topics. There is usually a presentation.


CU2600

"Come hang out and learn things about computers / security / cool other things."
This group is a little more random than DC217 but still has set topics and presentations.
